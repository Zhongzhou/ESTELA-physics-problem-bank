## Generating Images:

1. Open the "Convert.pptx" file. Select one image and right-click, Save as Picture.
2. Select file type as png, and modify the file name according to existing examples of images on the first page.
3. All images on the first page have been saved as png file. The "mass_far" group has already been created into multiple choice problems.
4. Every three images in a vertical column form a group, and can be uploaded to chatGPT to create problems following the prompts in the yaml file.
