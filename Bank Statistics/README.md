# Bank Statistics

This folder contains scripts and notebooks for analyzing the ESTELA physics problem bank.

## Setup

1. **Activate the virtual environment:**
   ```powershell
   .\venv\Scripts\Activate.ps1
   ```

2. **Install dependencies:**
   ```powershell
   pip install -r requirements.txt
   ```

## Scripts

- `tally_authors.py` - Extracts and tallies authors from all problem bank YAML files

## Usage

After activating the virtual environment, you can run scripts like:
```powershell
python tally_authors.py
```
